# Sight Translation Trainer

サイトトランスレーション英語練習PWAアプリ

## 使い方

1. `sight_translation_chunks.json` をアップロード
2. レッスンを選択
3. 日本語をスラッシュ区切りで見ながら英語に変換
4. 正解を確認・自己採点

## スマホにインストール

- **iPhone**: Safari で開く → 共有ボタン → 「ホーム画面に追加」
- **Android**: Chrome で開く → メニュー → 「ホーム画面に追加」

## ファイル構成

```
index.html      ← メインアプリ
manifest.json   ← PWA設定
sw.js           ← Service Worker（オフライン対応）
icon-192.png    ← アプリアイコン
icon-512.png    ← アプリアイコン（大）
```
